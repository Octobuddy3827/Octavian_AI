# 🧠 Octavian AI

> A better Perplexity. A better coding tool. Fully yours.

Octavian AI is an advanced AI assistant platform that connects to Google, OpenAI, Gemini, Pollinations AI, and NVIDIA NIM — all through a single, beautiful interface. Bring your own API keys and get a custom AI powerhouse.

---

## 🌟 Why Octavian AI?

Most AI tools lock you into one model. Octavian AI lets you:
- Switch between top providers instantly
- Use the best model for each task
- Keep your data — no third-party logging
- Customize everything

## ✨ Features

### 🎭 8 Specialized Agents

Each agent is fine-tuned for a different job:

- **General Assistant** — everyday queries
- **Code Master** — write, debug, and explain code
- **Research Expert** — find sources, synthesize information
- **Creative Writer** — drafts, stories, and copy
- **News Correspondent** — live headlines
- **Weather Expert** — forecasts and climate data
- **Market Analyst** — stocks and financial insights
- **Entertainment Guru** — movies, music, and recommendations

### 🔌 Supported Providers

| Provider | Models |
|---|---|
| OpenAI | GPT-4, GPT-4o, GPT-3.5 |
| Google | Gemini Pro, Gemini Flash |
| Anthropic | Claude 3 Opus, Sonnet, Haiku |
| NVIDIA NIM | Llama 3, Mixtral, and more |
| Pollinations AI | Free image & text generation |

### 🎨 Interface Highlights

- Real-time widgets: Weather · Stocks · News · Social
- Split sidebar + chat layout
- Dark / Light / Blue-Green themes
- Fully responsive
