## 🎨 Customization (Easy!)

### Change Colors (5 seconds)
Edit `styles.css` lines 8-9:
```css
--primary-blue: #2563eb;
--primary-green: #10b981;
```

### Add New Agent (2 minutes)
Edit `script.js` and add to `agents` object:
```javascript
myAgent: {
    name: 'My Agent',
    icon: 'fa-icon-name',
    description: 'What it does',
    systemPrompt: 'How it behaves'
}
```

### Change Welcome Message (1 minute)
Edit `script.js` agent greetings

### Modify Settings (1 minute)
Edit `config.js` configuration object
